Repositories over Reticulum

The git.quad4.io Gitea instance has been shut down. Source code is now hosted on RNGit over the Reticulum network.

You need Reticulum (RNS) installed with git-remote-rns to clone and push. See the Git Over Reticulum documentation for setup and usage.

Usage

Repository URLs use the rns:// scheme:

rns://06a54b505bb67b25ef3f8097e8001edc/public/<repo>

Clone example:

git clone rns://06a54b505bb67b25ef3f8097e8001edc/public/MeshChatX

Add a remote to an existing checkout:

git remote add origin rns://06a54b505bb67b25ef3f8097e8001edc/public/MeshChatX
git push -u origin main
